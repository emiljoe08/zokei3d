# Zokei3D.ai

**Photorealistic AI 3D Rendering Tool**

## Structure
```
/
├── index.html        → Landing page (zokei3d.ai)
├── app/
│   └── index.html   → Main app (zokei3d.ai/app)
```

## Stack
- Pure HTML/CSS/JS — no build step needed
- Supabase — authentication & render history
- FAL.ai — AI rendering (Flux Pro Kontext + Fill)
- Vercel — hosting & deployment
